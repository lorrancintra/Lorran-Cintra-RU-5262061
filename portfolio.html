Análise de Vendas com Excel (kaggle)

Neste projeto excel, escolhemos um banco de dados de vendas com 20 mil linhas. Apesar do Excel não ser a ferramenta ideal para esse volume de dados, o objetivo do projeto é mostrar como é possível realizar o tratamento, a organização e a apresentação das informações de forma clara e estruturada, superando limitações da ferramenta. Ao final, os dados são apresentados de maneira visual e acessível, facilitando a tomada de decisão por parte dos stakeholders.

Desafio do Titanic com Hybrid Ensemble (kaggle)

Neste projeto com aprendizado de máquina, trabalhamos com o clássico caso do Desastre do Titanic, um desafio introdutório amplamente utilizado para quem está começando na área de Machine Learning. Este projeto é uma excelente oportunidade para aplicar conceitos fundamentais e explorar diferentes técnicas de modelagem. Estou animado para acompanhar os resultados alcançados.

Atividade extensionista Uninter

Neste projeto, utilizando banco de dados e Power BI, selecionamos um setor voltado à saúde e à promoção da longevidade, abrangendo diferentes contextos e faixas etárias — desde adolescentes até idosos. O objetivo principal é desenvolver uma base de dados estruturada e dashboards interativos que permitam o monitoramento e a análise personalizada de treinos focados em saúde e bem-estar, oferecendo insights que apoiem a tomada de decisões e a evolução dos usuários.
